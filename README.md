# Calculadora de IMC-API
Projeto de calculadora de IMC para aula de API Development com Node.Js

Aplicação simples para cálculo de IMC com:

* Frontend em HTML, CSS e JS
* API em Node + Express

## Como rodar
```bash
npm install
node server.js
```
Frontend:
Abra `index.html` no navegador
API:
http://localhost:3000
